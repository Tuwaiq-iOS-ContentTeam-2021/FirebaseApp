# TimeLineApp
This is a timeline application were users can register and then login to post a text , each user has a name and a picture
 
 
 # What we're talking about?
TimeLine App allows you to express feeling , and Share writing about it with friends and family ! 

# Let's Dive into it, and see how it works..
Timeline App is developed by Swift 5.x and xCode 13.0.
 I use firebase (Firebase FireStore , Firebase Auth )
